# Careery
CareeryPlatform

Careery | Smart AI-Driven Hiring Platform AI & Deployment Lead (Graduation Project)

Project Overview: Developed an intelligent recruitment ecosystem that automates CV analysis, job matching, and interview preparation using state-of-the-art AI models.

LLM Fine-Tuning & CV Analysis:

Fine-tuned Llama 3-8B on a dataset of 30,000+ CVs to enhance performance in semantic analysis and professional career recommendations.

Implemented a system that extracts key skills, identifies weaknesses, and provides Keyword Optimization and career path guidance.

AI Interview Trainer (Computer Vision & Multimodal):

Integrated Qwen2-VL-2B-Instruct (Vision-Language Model) to analyze recorded interview videos.

Developed an automated reporting system evaluating Communication Clarity, Confidence, Body Language, Eye Contact, and Professionalism.

Provided users with actionable feedback and tips for improvement based on visual and vocal cues.

External Integrations: Leveraged multiple APIs to aggregate and recommend real-time Freelance projects and job listings based on the AI-generated job descriptions and candidate skills.
